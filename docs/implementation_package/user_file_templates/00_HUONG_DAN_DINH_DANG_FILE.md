# HUONG DAN DINH DANG FILE CHO NGUOI NHAP LIEU

## 1) Muc tieu
- Chuan hoa tai lieu truoc khi dua vao he thong chatbot phap ly.
- Giam loi OCR/RAG do cau truc tai lieu khong dong nhat.
- Dam bao tra cuu theo Dieu/Khoan/Chuong chinh xac.

## 2) Nguyen tac bat buoc
- Dinh dang de xu ly cuoi cung: `docx` (tu file goc pdf/doc/xlsx/image).
- Chi tap trung danh `Heading` dung cap cho tieu de va dau muc.
- Khong nhay cap heading (vi du: Heading 1 -> Heading 3).
- Moi tai lieu phai co metadata toi thieu truoc khi upload.

## 3) Metadata bat buoc khi upload
- `Tieu de tai lieu`
- `Loai tai lieu` (Quyet dinh/Quy dinh/Nhiem vu/Lich lam viec/Thuong)
- `Thu muc luu tru`
- `Ma vu viec` (neu co)
- `Do uu tien`
- `Nguoi upload`

## 4) Quy trinh lam viec de xuat
1. Chuan hoa file theo mau dung loai tai lieu.
2. Kiem tra Navigation Pane trong Word.
3. Doi chieu checklist truoc upload.
4. Upload file goc + file da chuan hoa (neu co).
5. Neu he thong bao loi, xu ly lai theo checklist.
